# Bloc de Notas online gratuito

Este es un bloc de notas gratuito creado en un simple y basico archivo HTML

### Funcionaliades

 - Italica    --> CTRL + I
 - Negrita    --> CTRL + B
 - Subrayado  --> CTRL + U
 - Guardado en el Local Storage
